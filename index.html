使用教程

详细的安装和使用教程，帮助您快速上手快捷魔方插件。

安装步骤

1

安装必要字体

下载并安装拆分笔画所需字体包，确保插件能正常拆分笔画。

下载字体包

安装提示：下载后解压字体包，双击打开字体文件，点击左上角"安装"按钮安装，按钮变灰色表示安装已完成。

2

安装运行环境（如需要）

如果系统是Win7，建议安装VSTO运行环境和.NET Framework 4.7.2。

Visual Studio Tools for Office Runtime .NET Framework 4.7.2

此步骤非必需，但如果您的系统非常老旧（如Win7），请优先考虑安装这两个运行环境。

3

打开安装包安装插件

打开"快捷魔方PPT.exe"或"快捷魔方Word.exe"安装包安装所需插件（建议鼠标右键以管理员身份运行安装包）。

安装完成后，打开对应的办公套件使用：
1. 安装PPT插件：打开PowerPoint或WPS演示文稿
2. 安装Word插件：打开Word或WPS文字

安装完成后，打开对应办公套件，会弹出询问"是否安装此自定义项"，点"安装"即可。

WPS启用插件

WPS默认禁用第三方插件，需要手动启用。请按照以下步骤操作：

1. 进入WPS文字和WPS演示文稿，点击左上角"文件"选项卡，往下找到并点击"选项"

2. 左侧顺下找到并点击"信任中心"，勾选"启用所有第三方COM加载项，重启WPS生效"，确定

截图仅以WPS文字为例，WPS演示文稿启用插件过程操作相同，不再赘述。 完成上述启用插件操作，关闭WPS，即可进入正式安装。

更新插件

打开"快捷魔方"功能区选项卡，找到并点击"检查更新"。如有新版本点击"立即更新"。

方法一：更新完成后，重启办公套件

如果你是WPS用户，则关闭WPS，重新打开WPS（WPS演示文稿或WPS文字）。

如果你是Office用户，则关闭Office，重新打开Office（PowerPoint或Word）。

方法二：更新完成后，通过COM加载项重启插件

基本市面上基于VSTO框架的插件的总开关都在COM加载项。因此，我们可以在不关闭或退出WPS和Office的情况下，也可以完成插件的重新加载。

如果你是WPS用户：

打开"工具"功能区选项卡，找到并点击"COM加载项"。

1. 先行禁用插件：取消勾选插件，点击确定。此时插件的功能区选项卡从功能区消失，即禁用插件。

2. 再行启用插件：禁用插件后，再次打开COM加载项，勾选插件，确定。此时插件重新出现在功能区上，即启用插件。

如果你是微软用户(MicroSoft Office用户)：

为方便管理插件，微软用户可在"功能区"先行启用"开发工具"选项卡。

1. 右键功能区空白处，选择"自定义功能区"。

2. 勾选"开发工具"，确定。

3. 打开"开发工具"功能区选项卡，找到并点击"COM加载项"，然后按照WPS用户的方法禁用再启用插件。

卸载插件

方法一：通过"已安装应用列表"卸载

与普通软件卸载一样，我们打开Windows菜单，打开系统设置，点击侧边"应用"， 打开"已安装应用列表"，搜索关键词"快捷"，右键或者鼠标移动到"•••"展开点击卸载。

方法二：通过插件安装目录下的安装器程序卸载

在安装目录文件夹找到并打开此安装器，点击卸载。

方法三：通过控制面板卸载

打开控制面板，找到"程序和功能"，在应用列表找到"快捷魔方"，鼠标右键卸载。
